MmDojoBundle
============

[![Build Status](https://secure.travis-ci.org/maastermedia/MmDojoBundle.png?branch=master)](http://travis-ci.org/maastermedia/MmDojoBundle)

Symfony2 bundle for Dojo Toolkit Javascript library (toolkit).

Features:

- dojo configuration
- twig extensions
